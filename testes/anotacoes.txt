server.py: Handles socket connections, client threads, and the game loop.
client.py: The Pygame frontend that connects to the server.
logic.py: Pure Python Uno rules (Deck, Validation, State) used by the server.
network.py: Protocol helper for sending/receiving JSON messages.
security.py: Handles simplified encryption (Caesar/XOR for demo or Fernet placeholders) and Auth tokens.
